***Файл .gitignore используется в системах контроля версий Git для указания файлов и каталогов,
которые должны игнорироваться при добавлении и коммите изменений в репозиторий.
Файл .gitignore является текстовым файлом, где каждую строку можно использовать для указания паттернов
на имена файлов и каталогов, которые следует игнорировать.
Файл .gitignore помогает поддерживать чистоту репозитория и облегчает совместную работу над проектом.
